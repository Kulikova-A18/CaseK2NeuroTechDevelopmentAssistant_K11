# CaseK2 NeuroTech Development Assistant (K11)

Проект представляет собой ИИ-ассистента для тимлидов и продактов, разработанный в рамках образовательного кейса «Нейротех». Система автоматизирует асинхронные ежедневные статус-обновления (daily) через мессенджер, интегрируется с задачными трекерами (например, Jira, Trello) и использует семантический анализ текста для выявления блокеров, оценки рисков срыва дедлайнов и прогнозирования прогресса спринта. Решение снижает когнитивную нагрузку на руководителей, устраняя необходимость ручного анализа статусов и ежедневных созвонов, особенно в распределённых командах с разными часовыми поясами.

### Инструкция по запуску

#### Вариант 1: Запуск вручную через Python  

1. Клонируйте репозиторий:

   ```bash
   git clone https://github.com/Kulikova-A18/CaseK2NeuroTechDevelopmentAssistant_K11.git
   cd CaseK2NeuroTechDevelopmentAssistant_K11
   ```

2. Установите зависимости:

   ```bash
   pip install -r requirements.txt
   ```

3. Запустите основной скрипт:

   ```bash
   python main.py
   ```

> Убедитесь, что у вас установлен Python ≥ 3.8.

#### Вариант 2: Запуск через Docker  

```bash
# Дайте права на выполнение
chmod +x docker-run.sh

# Запуск
./docker-run.sh up

# Пересборка и запуск
./docker-run.sh up true

# Проверка статуса
./docker-run.sh check

# Остановка
./docker-run.sh down
```

Мониторинг

```bash
# Просмотр использования ресурсов
docker stats

# Просмотр логов конкретного сервиса
docker-compose logs -f backend
docker-compose logs -f frontend
docker-compose logs -f telegram-bot
```

> Подробности смотри в `README-DOCKER.md`
